All Terminal Commands — Quick Reference

Keep this open during development and demo day

Step 1 — Start the Backend (FastAPI)
⚠ Open a NEW terminal window for this. The backend must stay running the whole time. Wait about 20 seconds after starting before trying to log in — the AI model needs time to load.
1
Go to backend folder
PS> cd C:\KCCSmartInfoX\backend
2
Activate the Python virtual environment
PS> venv\Scripts\activate
After running this, your terminal prompt will show (venv) at the start. That means it worked.
3
Start the backend server Keep this terminal open the whole time
(venv) PS> uvicorn main:app --reload --host 0.0.0.0 --port 8000
--host 0.0.0.0 → allows other devices on the same WiFi to connect to the backend
--reload → auto-restarts when you edit code (remove this for demo day)
Wait for: Application startup complete.
Step 2 — Start the Frontend (React/Vite)
⚠ Open a SECOND terminal window for this. Do not close the backend terminal.
4
Go to frontend folder
PS> cd C:\KCCSmartInfoX\frontend
5
Start the frontend dev server Keep this terminal open the whole time
PS> npm run dev -- --host
-- --host → exposes the frontend to your local network so phone and other laptop can access it
After starting, the terminal will show the Network URL — that's the URL for other devices.
Access URLs
ℹ After starting both servers, use these URLs:
Replace 192.168.0.106 with whatever IP the frontend terminal shows under "Network:"
This laptop (localhost)
http://localhost:5173
Landing page — use this on the admin laptop
Chat (this laptop)
http://localhost:5173/chat
Student chatbot — on this laptop
Admin Panel (this laptop)
http://localhost:5173/admin/login
Username: admin  |  Password: admin123
Other laptop / Phone (same WiFi)
http://192.168.0.106:5173/chat
Check current IP with: ipconfig — look for "IPv4 Address" under your WiFi adapter
Check Your Network IP (if URL above doesn't work)
i
Find your local network IP address
PS> ipconfig
Look for Wireless LAN adapter Wi-Fi → IPv4 Address.
It will look like 192.168.x.x. Use that IP in the URL above.
Install Dependencies (First Time / New Machine Only)
A
Install backend Python dependencies
PS> cd C:\KCCSmartInfoX\backend
PS> python -m venv venv
PS> venv\Scripts\activate
(venv) PS> pip install -r requirements.txt
This takes several minutes the first time — sentence-transformers is large (~300MB).
B
Install frontend Node.js dependencies
PS> cd C:\KCCSmartInfoX\frontend
PS> npm install
Git — Push Changes to GitHub
G
Save and push all changes to GitHub
PS> cd C:\KCCSmartInfoX
PS> git add .
PS> git commit -m "your message here"
PS> git push
G
Pull latest changes from GitHub (on second laptop)
PS> cd C:\KCCSmartInfoX
PS> git pull
Stop the Servers
X
Stop either server
PS> Ctrl + C
Press Ctrl + C in the terminal window running either the backend or frontend to stop it.
